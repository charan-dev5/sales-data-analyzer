# Sales Data Analyzer

A Python script that analyzes product sales data, calculates revenue, and exports a clean report to Excel.

## What it does
- Loads product sales and price data
- Calculates total revenue for each product
- Cleans data by removing zero-sales entries
- Identifies best seller and worst seller
- Exports final report to Excel

## Tools used
- Python
- Pandas
- openpyxl

## How to run
1. Install requirements: `pip install pandas openpyxl`
2. Run: `python data.py`
